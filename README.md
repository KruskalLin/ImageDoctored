# ImageDoctored

```shell
mkdir bounding_detect
mkdir configs
```
Directory structure:

area: xxx.jpg ...
class:
    a:
      a000.jpg
    	...
    b:
    ...
bounding_detect: xxx.jpg ...
configs: xxx.json ...


There are three types of manipulation: slicing, copy-move, removal. Run generate_configs() first then run them.
